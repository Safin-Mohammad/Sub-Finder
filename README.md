## Installation

For debian based linux os

```bash
sudo apt install python3 -y
git clone https://github.com/Safin-Mohammad/Sub-Finder
cd Sub-Finder
sudo apt install python3-pip
sudo pip3 install requests
sudo pip3 install threading 
python3 sub-finder.py
```
For termux

```bash
pkg install git
pkg install python3 -y
git clone https://github.com/Safin-Mohammad/Sub-Finder
cd Sub-Finder
pkg install python-pip
pip install requests
pip install threading 
python3 sub-finder.py
```

For arch based linux os

```bash
sudo pacman -Sy python3 -y
git clone https://github.com/Safin-Mohammad/Sub-Finder 
cd Sub-Finder 
pacman -S python-pip
sudo pacman -S python-requests
sudo pacman -S python-threading 
python3 sub-finder.py 
```
